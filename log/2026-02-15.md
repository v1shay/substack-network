12. **Layer stats: when to show L'(d), L''(d), r(d) in the chart**
    - Goal: L', L'', r at a depth are only meaningful once that layer is complete (e.g. layer 7 still being built ⇒ L'(7), L''(7), r(7) not meaningful). Table should show all depths; chart should hide incomplete.
    - Implementation: chart hides L', L'', r only for the **last** depth (the layer currently being built). All other depths show. No pending-count query; simpler rule. Table shows L', L'', r for every depth.
    - Removed the explanatory blurb from the layer stats page (avoid clutter).
    - docs/bfs.md: sentence that in the layer stats chart, L'(d), L''(d), and r(d) are shown for all depths except the last.
